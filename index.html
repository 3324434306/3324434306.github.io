FNF Healthbar 贴图设置器

使用 Lua 脚本为每个曲目单独设置自定义 healthbar 贴图 - 适用于 FNF 最新版本

效果预览

此预览展示自定义 healthbar 在游戏中的效果

确保贴图文件已放入 images/healthBar/ 目录

Lua 脚本

function onCreate() -- 创建自定义 healthbar makeLuaSprite('healthBarBG', 'healthBar/healthBarBG', 0, 0) setObjectCamera('healthBarBG', 'hud') addLuaSprite('healthBarBG', false) -- 使用自定义贴图 makeLuaSprite('healthBar', 'healthBar/healthBar_custom', 0, 0) setObjectCamera('healthBar', 'hud') addLuaSprite('healthBar', true) -- 设置图标位置 setProperty('iconP1.visible', false) setProperty('iconP2.visible', false) -- 创建自定义图标 makeLuaSprite('iconBF', 'icons/icon-bf', getProperty('iconP1.x'), getProperty('iconP1.y')) setObjectCamera('iconBF', 'hud') addLuaSprite('iconBF', true) makeLuaSprite('iconOpponent', 'icons/icon-opponent', getProperty('iconP2.x'), getProperty('iconP2.y')) setObjectCamera('iconOpponent', 'hud') addLuaSprite('iconOpponent', true) end function onUpdate() -- 更新 healthbar 位置 setProperty('healthBar.x', getProperty('healthBarBG.x')) setProperty('healthBar.y', getProperty('healthBarBG.y')) -- 设置 healthbar 缩放 setProperty('healthBar.scale.x', getProperty('health') * 0.5) end
